# ROSA — аудит сайта и идеи по росту продаж

**Объект:** rosa-villagio.ru — премиальный коттеджный посёлок на Новой Риге (Villagio Estate)
**Платформа сайта:** Tilda
**Продукт:** дома с участками от 17,3 млн ₽, три коллекции (95–580 м²)
**ЦА:** состоятельные семьи с детьми
**Дата:** май 2026

---

## 0. Главный вывод одной фразой

Сайт — красивая презентация, но **не касса**. Деньги теряются на двух вещах:
1. **Покупателю физически некуда оставить заявку** — единственная форма на сайте предназначена для агентств. Только телефон и Telegram.
2. **Нет механики мягкого захвата и подбора** — нет квиза, калькулятора, цен по планировкам, записи на просмотр. Ловится только самый «горячий» трафик, готовый звонить.

Эти правки + квиз-подборщик дадут больше, чем всё остальное вместе.

> **Примечание (исправление):** аналитика на сайте **есть и настроена хорошо** — Яндекс.Метрика с Вебвизором и e-commerce, GA4 + GTM, пиксели VK и Mail.ru. (В первичном беглом осмотре Метрика была ошибочно отмечена как отсутствующая — по реальному коду она на месте.)

## 2. Проблемы сайта (по приоритету)

### 🔴 Критично
1. **Нет формы захвата для покупателя ни на одной странице.** Единственная форма — «Уведомление для агентств». Покупатель может только позвонить/написать в WhatsApp → теряется весь, кто не готов к немедленному диалогу.
2. **Hero без захвата и без оффера.** Заголовок «Коттеджный посёлок ROSA… Официально. От застройщика» описывает, а не продаёт. На первом экране только телефон и ссылка на Telegram.
3. **Нет цен по планировкам** на страницах коллекций. Цена — вопрос №1; её отсутствие = отвал или «значит, дорого/непрозрачно».
4. **Цели/события в Метрике под формы.** Сама Метрика стоит (с Вебвизором), но нужно убедиться, что настроены цели на каждую форму/квиз — иначе не видно, какой канал приносит заявки.

### 🟠 Важно
5. **Нет соцдоказательств** — ни одного отзыва, истории резидента, лиц. Для премиума «кто мои соседи» — решающий вопрос.
6. **`© 2022` в футере** — выглядит заброшенным. Триггер недоверия.
7. **CTA только «Узнать подробнее»** — ведут на страницы без действия. Нет квиза, калькулятора, записи на просмотр.
8. **«Дом мечты за 71 900 ₽/мес»** — есть цифра, но статична, без интерактива и без кнопки.

### 🟡 Технет / SEO
9. **Нет alt у изображений** (сайт визуальный — минус в выдаче и доступности).
10. **Проверить скорость / Core Web Vitals** (тяжёлая графика, особенно мобайл).
11. Онлайн-чат отсутствует (WhatsApp/Telegram-виджет).

- **Пакет «дом под ключ + дизайн/отделка»** с фикс-ценой.

## 5. Идеи «сбоку» (каналы и рост вне сайта)

**Платный трафик (по данным рынка 2025–2026 работает лучше всего):**
- **VK Реклама, видеоформат** — на проектах загородки давала до 70% целевых лидов.
- **Яндекс ПромоСтраницы** — контентный «дожим», конверсия до ~58%.
- **Avito + Циан + Domclick** — обязательны; Avito снизил порог рекламы до 5 000 ₽.
- Геотаргет на дорогие районы Москвы и трафик конкурентов по Новой Риге; ретаргетинг.

**Контент и инфлюенс:** Telegram-канал посёлка, YouTube/Дзен (туры, интервью с архитекторами), блогеры (загородная жизнь, дизайн, семьи).

**Партнёрства и реферальность:** реферальная программа для резидентов («приведи соседа»), усиление агентской программы (прозрачная комиссия, CRM, материалы), партнёрства с дизайнерами интерьеров и частными школами.

**Офлайн:** дни открытых дверей и семейные бранчи на объекте, наружка на Новорижском шоссе, PR в Forbes Real Estate.
*(прим.: экскурсия с трансфером — отложена, есть организационные сложности)*

**Процессы (критично для дорогого товара):**
- **CRM + сквозная аналитика** — мерить стоимость лида И сделки по каналам.
- **Скорость ответа на лид** (первые минуты решают) + цепочки прогрева в email/Telegram под длинный цикл сделки.

## 6. Прототип

В папке `prototype/` — рабочий прототип-демо, собранный на фирменном визуале ROSA (логотип, палитра #051c2b / #ab182d, реальные фото). Демонстрирует механику, которой сейчас нет на сайте:
- квиз-подборщик (5 шагов → подбор коллекции → захват контакта);
- интерактивный калькулятор ипотеки (живой расчёт платежа);
- сравнение коллекций с ценами;
- формы захвата + чат-кнопки + триггер дефицита.

Открывается двойным кликом по `prototype/index.html`. Формы не отправляются на сервер (это демо); в рабочей версии подключаются к Telegram/CRM.

## 6.1. SEO и видимость в поиске

**Контекст:** в России ~половина+ поиска — **Яндекс**. Приоритет: сначала Яндекс, потом Google. SEO — игра вдолгую (месяцы); для быстрых продаж — платный трафик, SEO снижает стоимость лида со временем.

### ✅ Что уже хорошо (по реальному коду)
- Яндекс.Метрика с Вебвизором, картой кликов, e-commerce ✅
- GA4 + GTM, пиксели VK и Mail.ru ✅
- `canonical`, Open Graph, `lang="ru"` ✅
- `robots.txt` и `sitemap.xml` (с датами, обновлён 20.04.2026) ✅
- `title` и `description` заполнены ✅

### 🔴 Что чинить
1. **Нет семантических H1/H2** — текст свёрстан в `div`-блоки Tilda (0 тегов `<h1>`–`<h3>` в коде). → Назначить теги в Tilda: один H1 на страницу, H2 на секции.
2. **Нет микроразметки Schema.org / JSON-LD** (0 блоков). → Добавить `Organization`, `Product/Offer` с ценами, `BreadcrumbList`, `FAQPage`, `Residence` (через HTML-блок Tilda). Даёт расширенные сниппеты.
3. **Мусор в sitemap:** индексируются `/test`, `/rosa_old`, `/autumn-2022`, `/page53982397.html`. `/rosa_old` — риск дублей. → Убрать/закрыть.
4. **Хлам в аналитике:** два GTM-контейнера + мёртвый Universal Analytics (`UA-…`, отключён с июля 2023). → Почистить.
5. **alt у картинок** — ~4 из 15. Прописать.
6. **Скорость / Core Web Vitals** — HTML главной ~470 КБ + фото по 2–3 МБ. → PageSpeed Insights, сжать фото.
7. **title/description** заточить под гео + интент («купить», «Новая Рига», «Истра», «54 км от МКАД»).

### Что проверить в инструментах (нужен доступ владельца)
- **Яндекс.Вебмастер:** добавлен ли сайт, **выставлен ли регион (Москва и МО)** ← частая причина «нет в выдаче», ИКС, индексация, ошибки, отправлен ли sitemap.
- **Google Search Console:** индексация, Core Web Vitals, запросы.
- **PageSpeed Insights:** скорость моб/десктоп.
- **Яндекс.Бизнес:** карточка посёлка на Яндекс.Картах (локальный поиск).

### Типы запросов (под каждый — своя стратегия)
- **Брендовые** («коттеджный посёлок rosa») — быть №1 (проверить).
- **Гео + коммерция** («купить дом новорижское шоссе», «дома истринский район») — деньги, высокая конкуренция.
- **Информационные** («семейная ипотека на ИЖС») — контент/блог для прогрева.
