# ROSA — прототип конверсии + аудит

Демо-материалы для повышения продаж КП **ROSA** (rosa-villagio.ru).

- **`prototype/`** — рабочий прототип-демо (квиз-подборщик, калькулятор ипотеки, формы захвата) на фирменном визуале ROSA. Открывается в браузере.
- **`docs/АУДИТ-И-ИДЕИ.md`** — полный аудит сайта и список идей по росту продаж.

> ⚠️ Прототип на нейтральных/заглушечных текстах — демонстрирует механику, не финальный контент. Формы на сервер не отправляются. Логотип и фото взяты с текущего сайта, чтобы прототип выглядел «родным». Страница помечена `noindex` и не индексируется поисковиками.

После решения владельца сайта код переносится в текущую Tilda (вставкой в HTML-блок).
