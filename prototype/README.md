# Прототип ROSA — демо механики конверсии

## Как открыть
Дважды кликните по **`index.html`** — откроется в браузере. Интернет не нужен, всё работает локально.

## Что показывает
Прототип демонстрирует инструменты захвата заявок, которых **сейчас нет** на rosa-villagio.ru:

1. **Квиз-подборщик (премиум, инлайн)** — секция «Найдите свою коллекцию» прямо на странице (не модалка, без эмодзи). 4 вопроса → шаг-контакт (валидация +7, мессенджер) → подбор коллекции (Residents / Берёзовая / Луговая) по бюджету и числу людей. Кнопки «Подобрать дом» в шапке/hero/карточках скроллят к квизу.
2. **Калькулятор ипотеки (премиум)** — белый блок, главная цифра navy с анимацией. Кастомные слайдеры (стоимость, взнос), сегментированный выбор срока, 4 программы (Семейная 6% / IT 4,5% / Господдержка 8% / Базовая 18%), вывод платежа + суммы кредита + переплаты + срока в месяцах.
3. **Сравнение коллекций** с ценами «от».
4. **Формы захвата** + кнопки чата (WhatsApp/Telegram) + триггер дефицита «осталось N лотов».
5. **УТП «честная цена»** — полоса под hero (против конкурентов, прячущих цены).
6. **Условия покупки** — trade-in / рассрочка / ипотека программами (по итогам анализа конкурентов).
7. **Блок «Строит Villagio Estate»** — бренд и портфель сданных посёлков как опора доверия (вместо несуществующих «отзывов жителей» на старте продаж).
8. **Ход строительства** — таймлайн этапов + веб-камера (прозрачность).

## Важно
- Это **прототип на нейтральных/заглушечных текстах** — показываем механику и ощущение, а не финальный контент.
- **Формы не отправляются на сервер.** В рабочей версии заявки уходят в Telegram/CRM.
- Визуал (логотип, цвета, фото) взят с текущего сайта, чтобы прототип выглядел «родным».

## Папки
- `index.html` — сам прототип
- `img/` — логотип и фотографии (с текущего сайта)
- `shots/` — скриншоты для презентации
