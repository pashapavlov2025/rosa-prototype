# ТЗ: Премиальный калькулятор ипотеки для ROSA

## Контекст

ROSA — премиальный коттеджный посёлок на Новой Риге, средний чек 17–60 млн ₽. Сайт на Tilda, фирстиль: тёмно-синий `#051C2B`, винно-красный `#AB182D`, шрифт Formular (запасной — Inter).

## Задача

Сделать встраиваемый калькулятор ипотеки, который:
1. Помогает покупателю понять ежемесячный платёж и сумму переплаты.
2. **Закрывает конфликт «4,5% в одном блоке / 6% в другом»** через явный выбор программы. Пользователь выбирает свою ситуацию — видит свою ставку.
3. Ведёт к контакту с менеджером для уточнения условий.

## Что НЕ делать

- Типовой бухгалтерский «excel-в-вебе» калькулятор с десятком полей.
- Дефолтные браузерные слайдеры и select-ы — выглядит дёшево.
- Графики и пироги — не нужны, отвлекают.
- «Получите расчёт на e-mail» — это не лид-магнит, это рабочий инструмент на странице.
- Тяжёлые библиотеки (chart.js, jQuery и т.п.).

## Технические требования

- **Один HTML-файл** с inline CSS и vanilla JS.
- Никаких внешних зависимостей. Шрифт можно подключить с Google Fonts (Inter).
- Совместимо с T123-блоком Tilda.
- Размер итогового файла — до 25 КБ без шрифта.
- Mobile-first.
- Никаких `localStorage` / `sessionStorage`.

## Входные параметры

### Слайдер 1: Стоимость дома
- Диапазон: 15 000 000 — 60 000 000 ₽
- Шаг: 100 000 ₽
- Дефолт: 20 000 000 ₽

### Слайдер 2: Первоначальный взнос
- Диапазон: 15% — 50%
- Шаг: 1%
- Дефолт: 20%
- Под слайдером показывать абсолютное значение в ₽

### Селектор 3: Срок кредита
- Варианты: 5 / 10 / 15 / 20 / 25 / 30 лет
- Дефолт: 20 лет
- Внешне — горизонтальный сегментированный селектор (pill-buttons), не дропдаун

### Радио-карточки 4: Программа ипотеки
Четыре варианта, расположены в одну строку на desktop, 2×2 на mobile:

| Программа     | Ставка | Подпись                    |
|---------------|--------|----------------------------|
| Семейная      | 6%     | для семей с детьми         |
| IT            | 4,5%   | для IT-специалистов        |
| Господдержка  | 8%     | новостройки до 6 млн       |
| Базовая       | 18%    | без льготных условий       |

Дефолт: Семейная.

## Выходные значения

Главный блок (большая цифра, фокус всего экрана):

> **Ваш платёж по этой программе**
> **187 234 ₽ / мес**

Под главным блоком — приглушённой строкой:

- Сумма кредита: 16 000 000 ₽
- Переплата за весь срок: 28 936 160 ₽
- Срок: 240 месяцев

CTA внизу:

> **[Обсудить условия с менеджером]**

(На этапе прототипа: `console.log(state)` и перехвачен клик.)

## Формула

Аннуитетная (стандарт российской ипотеки):

```
P = S × r × (1 + r)^n / ((1 + r)^n - 1)
```

где:
- `P` — ежемесячный платёж
- `S` — сумма кредита (стоимость − первоначальный взнос)
- `r` — месячная ставка (годовая ÷ 12 ÷ 100)
- `n` — количество месяцев (срок в годах × 12)

Edge cases:
- Если ставка = 0% → `P = S / n` (защита от деления на ноль)
- Округление до целых рублей

## Дизайн-гайд

### Цвета
- Фон: `#FFFFFF`
- Текст основной: `#1A1A1A`
- Текст приглушённый: `#6B7280`
- Акцент (выбранная программа, активный сегмент): `#051C2B`
- Граница неактивная: `#E5E7EB`
- Главная цифра — тоже `#051C2B` (не яркий цвет, premium feel)

### Типографика (Inter)
- Главная цифра платежа: 48–64px desktop / 36–44px mobile, weight 600, tabular-nums
- Лейблы слайдеров: 14px, weight 500, uppercase, letter-spacing 0.5px
- Значения под слайдером: 18–20px, weight 500
- Body / подписи программ: 13–14px, weight 400

### Слайдеры — кастомные

```css
/* WebKit + Mozilla стилизация range input */
input[type=range]::-webkit-slider-thumb {
  -webkit-appearance: none;
  width: 24px; height: 24px;
  background: #051C2B;
  border-radius: 50%;
  cursor: pointer;
  border: 3px solid #FFFFFF;
  box-shadow: 0 0 0 1px #051C2B, 0 2px 8px rgba(5,28,43,0.15);
}
input[type=range]::-webkit-slider-runnable-track {
  height: 4px;
  background: linear-gradient(to right, #051C2B var(--fill, 50%), #E5E7EB var(--fill, 50%));
  border-radius: 2px;
}
```

Заливка слайдера (часть до thumb) рисуется через `--fill` CSS-переменную, которую JS обновляет на каждое `input`.

### Сегментированный селектор (срок)
- Контейнер с тонкой границей `#E5E7EB`, скруглением 8px
- Каждый сегмент — кнопка, hover показывает `#F7F8FA` подложку
- Активный сегмент — navy фон `#051C2B`, белый текст
- Переход 150мс

### Радио-карточки (программа)
- Карточка: padding 16–20px, border 1px `#E5E7EB`, скругление 12px
- Hover: граница темнеет до `#9CA3AF`
- Выбрана: граница 2px `#051C2B`, очень лёгкая навиграя подложка
- Внутри: название программы (16px weight 600), ставка в правом углу или под названием (20px weight 600), подпись приглушённая (13px)

### Анимация чисел
- При изменении любого параметра — плавная анимация главной цифры платежа через requestAnimationFrame
- Длительность: 300мс
- Easing: ease-out
- Tabular-nums чтобы цифры не «прыгали» по ширине

## Форматирование чисел

- Тысячи разделять неразрывным пробелом: `20 000 000 ₽`
- Использовать `Intl.NumberFormat('ru-RU')` или ручной разделитель
- Никаких «20кк», «20М» и подобного — это премиум

## Что должно получиться

Один HTML-файл, который:
- Открывается в браузере, демка работает мгновенно
- Реагирует на любое движение слайдера / клик мгновенно (без debounce задержек на UI)
- Главная цифра анимируется при изменении параметра
- На мобайле слайдеры удобно двигать пальцем
- Копируется в Tilda T123 и сразу работает
- Выглядит так, что хочется его потрогать
